## 安全加固（ikol + claw-wiki）

### 配置层（openclaw.json）
```json
"tools": {
  "deny": ["gateway", "exec"],
  "fs": { "workspaceOnly": true }
},
"sandbox": {
  "workspaceRoot": "/root/.openclaw/workspace/<agent>"
}
```
- `exec` 完全禁止（shell 命令，规避 /tmp 写入等绕过问题）
- `gateway` 完全禁止（无法改配置/重启服务）
- `fs.workspaceOnly: true` 文件读写锁定在各自 workspace
