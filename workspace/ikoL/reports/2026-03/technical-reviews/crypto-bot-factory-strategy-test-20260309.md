## TC-1：保守稳健型

### 自然语言输入
> "保守稳健策略，趋势为主兼顾均值回归，双向交易但不偏向任何一侧。杠杆3倍，每笔只用5%资金，止盈止损宽松一点（止盈RR=3.5），对行情阶段很敏感，震荡市就少交易。不开滚仓，保住本金优先。"

### 参数解读
| 参数组 | 配置 |
|--------|------|
| 信号权重 | 趋势(40%) > 均值回归(20%) > 动量(20%) > 量能(10%) > 波动率(10%) |
| 方向 | 多空双向 (long_bias=0.50) |
| 杠杆 | 3x（最大5x） |
| 入场阈值 | 0.35（保守，少交易） |
| 风控 | 止损2.5ATR / 止盈RR=3.5 / 移动止损开启 |
| 仓位 | 每笔5%，上限25% |
| 滚仓 | 关闭 |
| 行情敏感度 | 0.70（高，震荡市自动减少交易） |

### 回测结果
| 指标 | 数值 |
|------|------|
| 总收益率 | **-0.8%** |
| Sharpe | -0.21 |
| 最大回撤 | 3.3% ✅ |
| 交易次数 | 73笔（低频）|
| 胜率 | 35.6% |
| 盈利因子 | 0.92 |
| 止盈/止损/爆仓 | 6 / 47 / 0 |
| 多/空笔数 | 27 / 46 |
| 最终资金 | $9,918 |

### 分析
- ✅ **回撤控制极佳**：3.3%，是所有策略中最低的
- ✅ **无爆仓**：低杠杆 + 高行情敏感度有效防止极端损失
- ⚠️ **收益接近0**：保守参数在熊市中几乎不赚钱，止盈只触发6次
- ❌ **胜率偏低**：35.6%，入场信号质量有待提升
- **结论**：适合风险厌恶型，亏不多也赚不多，熊市保本

## 综合对比分析

### 行情适配性
| 行情类型 | 最适策略 | 原因 |
|---------|---------|------|
| 单边下跌（熊市） | TC-3（纯空） | 方向正确，滚仓放大下跌收益 |
| 震荡盘整 | TC-4（均值回归） | 低回撤，快进快出 |
| 趋势行情（双向） | TC-5（动量+滚仓） | 大行情时滚仓爆发 |
| 不确定/保本 | TC-1（保守） | 回撤最小，不赚不亏 |
| **绝对避免** | TC-2（激进偏多） | 熊市偏多=爆仓风险 |

### 关键发现
1. **滚仓是核心差异因子**：TC-5（开滚仓）+17.1% vs 同参数但无滚仓的效果（参考基线+9.9%）
2. **方向判断比参数调优更重要**：TC-3纯空+8.2%，TC-2偏多-34.5%，差距完全来自方向
3. **杠杆与行情波动率需要匹配**：熊市高杠杆偏多 = 灾难
4. **高胜率≠高收益**：TC-4胜率37.2%但只赚+1.6%；TC-5胜率30.6%却赚+17.1%（赢大亏小更重要）

### Bug发现（延续自动化测试结论）
| Bug | 严重级别 | 描述 |
|-----|---------|------|
| 战术参数漂移超30%未被截断 | 🔴 高 | entry_threshold 进化时漂移36.4%，代码未强制钳制 |
| fetch_data.py 未验证天数上限 | 🟡 中 | days=366 正常执行，安全护栏失效 |
| 数据不足一段时无提示降级 | 🟢 低 | 静默执行单段回测，用户体验不佳 |
