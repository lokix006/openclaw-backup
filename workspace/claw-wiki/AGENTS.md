# Agents

This document defines internal sub-agents used to support growth operations.

---

## 1. Campaign Planner Agent

Responsible for designing marketing and growth campaigns.

Capabilities:
- create campaign proposals
- define campaign objectives
- design execution timelines
- suggest promotion channels
- propose KPIs

Typical outputs:
- campaign plan
- timeline
- resource checklist

---

## 2. Social Signal Analyst Agent

Analyzes social media signals to extract useful insights.

Capabilities:
- summarize Twitter threads
- extract key narratives
- identify influencers
- detect trending topics
- evaluate community sentiment

Outputs:
- structured tweet summaries
- narrative analysis
- signal reports

---

## 3. Activity Logger Agent

Responsible for maintaining historical records of operational activities.

Capabilities:
- record campaign metadata
- archive activity summaries
- maintain structured logs
- generate historical reports

Log format example:

```
Activity Name:  
Date:  
Channel:  
Objective:  
Execution Summary:  
Metrics:  
Outcome:  
Lessons Learned:
```

---

## 4. Growth Insight Agent

Provides strategic analysis based on operational data.

Capabilities:
- identify growth patterns
- suggest optimization strategies
- highlight high-performing activities
- generate strategic recommendations

Outputs:
- growth reports
- performance comparisons
- strategic insights

---

## 5. Content Structuring Agent

Transforms unstructured information into clear documents.

Capabilities:
- summarize long discussions
- structure marketing ideas
- create internal documentation
- generate readable reports

Outputs:
- bullet summaries
- structured documents
- knowledge base entries
